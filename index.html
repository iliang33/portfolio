Album Review App

Technologies used: Java

Description: This is an application that allows one to organize and track their thoughts on albums they have listened to. One can create album review entries with a name, artist, genre, rating, and review. After creating an entry, one may add a tracklist of songs to it while giving each song a rating. For organization, the app also allows for the creation of categories that one can add existing entries to, as well as have the options to sort created entires by name, artist, or rating
